This is the basic look-and-feel for websites for the Early Detection Research
Network (EDRN_), which loosely follows the style of the National Cancer
Institute (NCI_).  It was developed by the Informatics Center (IC_), operated
by JPL_.

Although intended for the EDRN public portal, this package may be used in any
Plone_ 3 (or later) site.
